# 3쿠션 길보기 트레이너

3쿠션 당구의 핵심 능력인 **길보기(경로 예측)**를 게임처럼 반복 훈련할 수 있는 웹 기반 학습 서비스입니다.

**[바로 시작하기 →](https://billiards-trainer.vercel.app)**

## Features

- **물리 시뮬레이션** — 공 이동, 쿠션 반사, 공-공 충돌, 마찰, 스핀(회전) 효과
- **문제 자동 생성** — Mirror Reflection(거울 반사) 기하학 기반 유효한 3쿠션 경로 생성
- **조준 시스템** — 수구(흰공) 드래그로 방향 조준
- **타점(스핀) 패드** — 밀어치기, 끌어치기, 좌/우 회전 설정
- **강도 슬라이더** — 샷 파워 조절
- **경로 시각화** — 샷 후 실제 경로(파란선)와 정답 경로(초록 점선) 비교
- **정답 보기** — 정답 경로 + 강도/타점/각도 표시, 정답 자동 적용 기능
- **채점 시스템** — 정확도, 시간, 시도 횟수 기반 점수
- **5단계 레벨** — 기본 앞돌리기 → 두께 조절 → 강도 조절 → 회전 → 복합
- **로컬 저장** — 진행도와 점수 localStorage 저장
- **모바일 지원** — 터치 조작 대응, 반응형 레이아웃

## 게임 방법

1. 레벨을 선택합니다
2. 당구대 위의 공 배치를 확인합니다 (흰공 = 수구, 노란공 = 제1목적구, 빨간공 = 제2목적구)
3. **수구를 드래그**하여 방향을 조준합니다
4. **강도**와 **타점(스핀)**을 설정합니다
5. **샷!** 버튼을 눌러 실행합니다
6. 수구가 제1목적구를 맞힌 뒤, **3개 이상의 쿠션**을 거쳐 제2목적구를 맞히면 성공!

## 문제 생성 원리

**Mirror Reflection (언폴딩) 방식**으로 유효한 3쿠션 경로를 생성합니다:

1. 쿠션 시퀀스 선택 (예: 상단 → 우측 → 하단)
2. 목표점을 쿠션별로 역순 반사하여 테이블을 "펼침"
3. 직선 경로를 다시 접어 실제 쿠션 접촉점 계산
4. Forward Simulation으로 물리 엔진 검증
5. 레벨별 난이도에 맞는 배치만 채택

## Getting Started

```bash
npm install
npm run dev
```

`http://localhost:5173` 에서 실행됩니다.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Zustand (상태 관리)
- Canvas API (렌더링)
- localStorage (진행도 저장)

## 프로젝트 구조

```
src/
├── physics.ts          # 물리 엔진 (충돌, 반사, 마찰, 스핀)
├── problems.ts         # 문제 생성 (Mirror Reflection + 시뮬레이션 검증)
├── store.ts            # 게임 상태 + 진행도 저장 (Zustand)
├── vector.ts           # 2D 벡터 유틸리티
├── constants.ts        # 테이블/물리 상수
├── types.ts            # 타입 정의
├── App.tsx             # 메인 앱
└── components/
    ├── BilliardCanvas.tsx  # Canvas 렌더링 + 마우스/터치 조작
    ├── ShotControls.tsx    # 강도/타점/샷 버튼
    ├── GameScreen.tsx      # 게임 화면 (캔버스 + 컨트롤 + 결과)
    ├── HomeScreen.tsx      # 홈 화면
    └── LevelSelect.tsx     # 레벨 선택
```
